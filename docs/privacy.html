プライバシーポリシー / Privacy Policy

最終更新: 2026年4月29日 / Last updated: April 29, 2026

1. 取り扱う情報

hyakkirodoku (以下「本アプリ」) は、開発者個人のローカル環境で動作するツールです。 本アプリは TikTok ログイン Kit および Content Posting API を利用するため、以下の情報を扱います。

  • TikTok アクセストークン (OAuth 認証で取得、ローカルファイル data/tiktok_token.json に保存)
  • 認証済み TikTok アカウントの公開ユーザー名 (UI 表示用に user.info.basic スコープで取得)

2. 情報の利用目的

取得した情報は、開発者本人のアカウントへ生成済み動画を Inbox (ドラフト) としてアップロードする目的にのみ使用されます。 第三者への提供・販売・分析等の目的では一切利用しません。

3. 情報の保管

すべての情報は本アプリの動作する開発者のローカルマシン内にのみ保存されます。 外部サーバーへの送信・収集は行いません (TikTok API への通信を除く)。

4. 第三者提供

本アプリは取得した情報を第三者に提供することはありません。

5. 情報の削除

ユーザー (開発者本人) は、ローカルの data/tiktok_token.json を削除することで、保存されているアクセストークンを完全に削除できます。 TikTok アカウント側からの認可取り消しは TikTok のアプリ管理画面から行えます。

6. 改定

本ポリシーは予告なく改定される場合があります。改定後は本ページに掲載された時点で効力を生じます。

7. お問い合わせ

本ポリシーに関するお問い合わせは、本アプリの GitHub リポジトリの Issue 機能をご利用ください。

Privacy Policy (English)

1. Information We Handle

hyakkirodoku (the "App") runs locally in the developer's personal environment. Because the App uses the TikTok Login Kit and Content Posting API, it handles the following information:

  • TikTok access token (obtained via OAuth, stored locally in data/tiktok_token.json)
  • Public TikTok username (obtained via user.info.basic scope, used only for in-app display)

2. Purpose of Use

The information is used solely to upload generated videos as Inbox (draft) to the developer's own TikTok account. We do not share, sell, or analyze information for any third-party purpose.

3. Data Storage

All information is stored only on the developer's local machine where the App runs. We do not transmit or collect data to any external server (other than communications with the TikTok API).

4. Third-Party Disclosure

The App does not disclose obtained information to any third party.

5. Deletion

The user (the developer) can fully delete the stored access token by removing the local file data/tiktok_token.json. Authorization can also be revoked from the user's TikTok account management screen.

6. Updates

This policy may be updated without prior notice. Updates take effect when posted on this page.

7. Contact

For inquiries regarding this policy, please use the Issue tracker of the App's GitHub repository.
